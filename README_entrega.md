# Entrega aprendizaje supervisado - Titanic

Este paquete contiene notebooks para la práctica de aprendizaje supervisado con el conjunto de datos del Titanic.

Archivos incluidos:

- 1_clean.ipynb
- 2_EDA.ipynb
- 3_feature_engineering.ipynb
- 4_machine_learning.ipynb
- 5_pipeline.ipynb

Los notebooks incluyen limpieza de datos, análisis exploratorio, ingeniería de características, entrenamiento de modelos y pipeline de Machine Learning.
